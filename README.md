# This is a CD Management Application

## Ruby version

Ruby-2.2.3

## API Docs

[CD Mam API Documentation](https://lit-wildwood-7387.herokuapp.com/api/docs)

## Installing 
```sh
  git clone git@github.com:djtsheps/CDMan.git 
  
  cd CDMan
  
  bundle install
  
  rake db:migrate

  rake db:seed
```
## Running tests
```sh
  bundle exec rake
```
